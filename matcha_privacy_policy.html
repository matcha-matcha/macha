MATCHA

Privacy Policy

Last updated: June 2025

🔒 MATCHA does not collect, store, or transmit any personal data to external servers.

Overview

MATCHA ("the Extension") is a Chrome extension developed by matcha that automatically sets YouTube subtitles to the user's preferred language. This Privacy Policy explains how the Extension handles user data.

Data We Collect

MATCHA collects only the following data, stored locally on your device using Chrome's built-in storage:

  • Subtitle language preference — the language you select in the extension popup
  • Subtitle on/off setting — whether the automatic subtitle feature is enabled

This data is stored using chrome.storage.sync, which may sync across your Chrome-signed devices via your Google account. No data is sent to MATCHA's servers.

Data We Do NOT Collect

  • Browsing history or YouTube watch history
  • Personal information (name, email, location, etc.)
  • Video content or subtitles
  • Usage statistics or analytics
  • Any data transmitted to external servers

Permissions Used

  • activeTab — to interact with the YouTube tab you are currently viewing
  • storage — to save your language preference locally
  • host_permissions (youtube.com) — to run the extension only on YouTube

Third-Party Services

MATCHA operates entirely within YouTube's existing subtitle system (Auto-translate). It does not use any third-party translation APIs or external services.

Changes to This Policy

If this Privacy Policy is updated, the new version will be published on this page with a revised date.

MATCHA

プライバシーポリシー

最終更新日：2025年6月

🔒 MATCHAは個人データを外部サーバーに収集・送信しません。

概要

MATCHA（以下「本拡張機能」）は、YouTubeの字幕を自動で設定するChrome拡張機能です。本ポリシーでは、ユーザーデータの取り扱いについて説明します。

収集するデータ

MATCHAはChromeの組み込みストレージを使用して、以下のデータのみをデバイス内に保存します：

  • 字幕言語設定 — ポップアップで選択した言語
  • 字幕オン/オフ設定 — 自動字幕機能の有効/無効状態

データは chrome.storage.sync を使用して保存され、Googleアカウントでサインイン中の場合、Chrome同期によって他のデバイスと共有される場合があります。MATCHAのサーバーへのデータ送信は一切ありません。

収集しないデータ

  • 閲覧履歴・YouTube視聴履歴
  • 個人情報（氏名・メールアドレス・位置情報など）
  • 動画コンテンツや字幕の内容
  • 利用統計・アナリティクス
  • 外部サーバーへ送信されるいかなるデータ

使用するパーミッション

  • activeTab — 現在開いているYouTubeタブを操作するため
  • storage — 言語設定をローカルに保存するため
  • host_permissions (youtube.com) — YouTube上でのみ動作させるため

サードパーティサービス

MATCHAはYouTubeが提供する既存の自動翻訳機能のみを使用します。外部の翻訳APIや外部サービスは一切使用しません。

ポリシーの変更

本ポリシーを更新する場合は、このページに新しいバージョンを掲載し、更新日を改訂します。
